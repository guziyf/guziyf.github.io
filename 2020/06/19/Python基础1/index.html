发表于 2020-06-19 分类于 Python

根据阿里云大学提供的课程（尚学堂网课）整理了如下基础知识，基础知识学习路线部分一共分了三节大课，统一整理，方便以后回顾。

第一部分主要介绍语法基础，包括一个质数练习的优化

一、语法入门

1、几个概念

  1. 表达式
    如：10 + 5 8 - 4
    在交互模式输入一个表达式，解释器会自动将表达式结果输出
  2. 语句
    如：print()
    a = 10
    执行结果对程序产生影响
  3. 程序
    由语句和表达式构成、
  4. 函数
    1. 内置函数
      • 由Python解释器提供的，直接使用
    2. 自定义函数
      • 由程序员自主创建

2、基本语法

  1. 严格区分大小写

  2. 每一行就是一条语句，每条语句以换行结束

  3. 规范中建议每行不超过80个字符

  4. 一条语句可以分多行编写，以“\”结尾

  5. 缩进严格，不要随便缩进！

  6. 用# 来注释，一般在# 后跟一个空格

3、字面量和变量

  • 字面量，如：1，2，3，4，5，‘Hello’
  • 变量，如 a = 1

4、标识符

  1. 可以含有字母、数字、_ ，但不能数字开头

    例如：a_1, _a1, _1a, 但不能是 1a

  2. 不能是Python中的保留字和关键字

二、数据类型

1、数值

  1. 整数、浮点数、复数

  2. 所有整数都是int类型，整数可以无限大，如：

    a = 1932192389138913819381938913891

    所有小数都是float类型，如：

    a = 12.12312312313

  3. 数值过长，可以用下划线分隔

    a = 123123_23131_32131

  4. 二进制：0b开头，八进制：0o开头，十六进制：0x开头，print打印时都以十进制显示

  5. 字符串类型

    1. 可以用‘ ’也可用“ ”，但不能混用

    2. 使用’’’ ‘’’时回保留换行格式

  6. 转义字符“\”

三、运算符

1、算术运算符

  1. +、-、*、/

  2. +如果是字符串则进行拼串操作 a = ‘ad’+’ ‘+’asda’，不支持减法操作

  3. a = 5 - True可以，a = 4

  4. ‘/’运算时总会返回一个浮点类型，不能除以0

  5. ‘//’整除，只保留计算后的整数位

  6. ‘**’幂运算

  7. ‘%’模运算，结果取余

2、赋值运算符

+=、-=、=、*=、/=、//=、%=

对浮点数做整除运算结果也会返回一个浮点数

3、关系运算符

  1. 总会返回一个布尔值

  2. 同C语言，但会返回结果

  3. is,is not

4、逻辑运算符

  1. not、and、or

  2. a = not a 非运算取反操作，结果只返回True和False

  3. and与运算 a = True and False ,返回False

  4. a = 1 and 2 ， 返回 2 ，因为第一个是True，所以直接返回第二个值（与or相反）

  5. ```python
    a = 1 < 2 < 3 # 相当于1 < 2 and 2 < 3，返回True
    a = 10 < 20 > 15 # 20 > 10 and 20 > 15 ，返回True

    and比or高，not比and高······
