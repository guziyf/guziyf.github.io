二刷C语言

发表于 2020-06-18 分类于 C语言

通过mooc大学翁恺老师的C语言程序设计课程，再次巩固学习C语言。

第1章：程序设计与C语言

1.计算机和编程语言

  • 程序使用特殊的编程语言写出来表达如何解决问题的
  • 不是用编程语言来和计算机交流，而是描述要求它如何做事情的过程或方法
  • 要让计算机做计算，就要找出计算的步骤，然后用编程语言写出来
  • 计算机做的所有的事情都叫计算（如：看一个电影，画一个图。都是计算）
  • 计算的步骤就是算法
  • 程序的执行
    1. 解释：借助一个程序，那个程序能试图理解你的程序，然后按照你的要求执行
    2. 编译：把程序翻译成计算机真正能懂的语言——机器语言，然后就能执行
  • 任何语言既可以解释执行，也可以编译执行，具体不看语言，而看编译器的实现
    1. 编译型语言有确定的运算性能
    2. 解释型语言有特殊的计算能力

2.C语言

  • 1973年3月，第三版的Unix上出现了C语言的编译器
  • 1973年11月，第四版Unix是完全由C语言重写的
  • 1989年ANSI发布了一个标准——ANSI C
  • 1990年ISO接收了ANSI的标准——C89
  • 1995年和1999年两次更新——C95 和 C99
  • 目前最新的版本是2011年的——C11（目前C99和C11有99.5%都相同，除了gets()函数不可能，基本不影响）

3.第一个程序

一个C语言学习的在线开发工具：https://clin.icourse163.org/ 可以方便的进行单步调试

第2章：计算

1.变量

  • int a = 0变量名是a，类型是int，初始值是0。变量定义：<类型名称><变量名称>

  • 变量的名字是一种“标识符”，只能由字母、数字和下划线组成，数字不可以出现在第一个位置

  • “=”赋值运算符，在计算机中是一个动作，关系是静态的，动作的动态的

  • 所有变量第一次被使用前，都应该初始化

  • 组合变量定义：int a = 0，b = 100；

  • C语言是一种有类型的语言，所有的变量在使用前必须定义或声明，所有变量必须具有确定的数据类型。

  • scanf输入的默认规则

  • 固定不变的数，是常数，写在程序；里称作直接量，最好的使用方式是定义一个常量：const int A = 100；（C99写法），使用有意义的名字的常量便于程序员读，使用大写

2.数据类型

  • 整数，
    1. int a
    2. printf(“%d”,a)
    3. scanf(“%d”,&a)
  • 浮点数
    1. double b
    2. printf(“%f”,b)
    3. scanf(“%lf”,&b)

3.表达式

  • 单目运算，如a*-b，运算a-b，因为-号是单目运算符

  • 交换a，b(固定的套路)

  • +=、-=、*=、/=、%=

  • ++ 、 —

  • a++是a+1以前的值，++a是a+1以后的值

第3章：判断与循环

  • 关系运算符：==、!=、>、>=、<、<=，连续关系从左到右进行
  • do-while和while循环的区别在循环体执行结束时判断条件，也就是说循环至少执行一遍，然后再判断条件。
  • 小套路：求和时，记录结果的变量应该初始化为0，而做求积的变量时，记录结果的变量应该初始化为1

第4章：循环控制

  • goto <标号>，类似汇编语言的跳转语句，会跳到 <标号>[:]

第5章：进一步判断和循环

第6章：数组与函数

1.数组

  • 是一种容器，特点：
    1. 其中所有的元素具有相同的数据类型
    2. 一旦创建，不能改变大小
    3. *（数组中的元素在内存中是连续依次排列的）
  • C99的特性，定义数组时，下标定义时的值可以是动态的，用a[n]，先定义n是多少

2.函数的定义与使用

就那样

3.函数的参数和变量

  • 注意参数之间要用“,”分割
  • void 函数名，不能使用带值的return，可以没有return，调用时不能做返回值的赋值
  • 如果把函数体放在主函数下面，必须要在主函数上面进行声明（写函数原型），声明内容和函数名一致并加上“；”，但是参数的名字可以不一样。
  • C语言在函数传参时，会自己进行类型不匹配的转换
  • 形式参数和实际参数的叫法是一种古老的方式，已经过时，在称呼时，认为它们是参数和值的关系
  • 一个块中可以嵌套大括号，同一个块中不能定义同名变量
  • mac中用echo $?可以访问main函数中的return返回的值

4.二维数组

  • int a[3] [5];//行是竖起来，列是横起来
a[0] [0] a[0] [1] a[0] [2] a[0] [3] a[0] [4]
a[1] [0] a[1] [1] a[1] [2] a[1] [3] a[1] [4]
a[2] [0] a[2] [1] a[2] [2] a[2] [3] a[1] [4]
  • 列数不可以省略，函数可以由编译器来数，每行一个{}，逗号分隔

第8章：指针与字符串

1.指针

  • 运算符&

    • 取得变量的地址

    • 每一个变量都是放在地址里的

    • ```c

      include

      int main(int argc, char const *argv[])
      {

      int i = 0;
printf("0x%x\n",&i);             //0xe85436bc        在64位架构里不一样，int4位，地址8位
printf("%p\n",&i);               //0x7ffee85436bc   
return 0;

      }

    • 数组地址之间的差距永远是4

  • 指针

    1. 就是保存地址的变量

    2. p指向i，实际就是说p里面的值是i的地址

    3. 没有int*这种类型

    4. 指针变量

      • 指针变量的值是具有实际值的普通变量的地址

    5. 作为参数的指针

    6. 数组传递时，传递为指针void max(int a[])和void max(int *a)作为函数原型是等价的，数组变量是特殊的指针

    7. 数组变量是const的值，所以不能用b = a

      • 可以看作int b[] —> int * const b，可以被看作，但不等于“就是”

2.字符类型

  • char是一种整数，也是一种特殊的类型：字符。

  • 用单引号白哦是字符字面量：‘a’,’1’

  • ‘’也是一个字符

  • printf和scanf里面用%c来输入输出字符

  • 讲了一下ASC||编码，比如’1’的编码是49

  • 逃逸字符

    • 由一个“\”开头，后面跟上另一个字符，这两个字符合起来，组成了一个字符

      | 字符 | 意义 | 字符 | 意义 |
      | —— | ——————— | —— | ————— |
      | \b | 回退一个 | \ “ | 双引号 |
      | \t | 到下一个表格位 | \ ‘ | 单引号 |
      | \n | 换行 | \ \ | 反斜杠本身 |
      | \r | 回车 | | |

3.字符串

  • 0或’\0’是一样的，但和’0’不同

  • 0标志字符串的结束，但它不是字符串的一部分

  • 计算字符串长度的时候不包含这个0
  • 字符串以数组的形式存在，以数组或指针的形式访问
  • string.h里有很多处理字符串的函数
  • 字符串变量：
    • char* str = “Hello”;
    • char word[] = “Hello”;
    • char line[10] = “Hello”
    • 这个数组的长度是6，结尾还有表示结束的0
    • 编译器会自动帮你生成结尾的0
  • 意识流：很小的地址在代码段，只读的，如果用S[0] = ‘B’写的话会报错
  • 数组（char word[] = “Hello”）：作为本地变量空间自动被回收
  • 指针（char* str = “Hello”）：“这个字符串不知道在哪”，动态分配空间

4.字符串计算

  • char buffer[100] = “”;这是一个空的字符串,buffer[0] == ‘\0’
  • char buffer[] = “”;这个数组的长度只有1

string.h

  • strlen返回s的字符串长度（不包括结尾的0）

  • strcmp

  • strcpy拷贝字符串，把s2拷贝到s

  • strcat连接，把s2拷贝到s后面，接成一个长字符串，返回s，s必须具有足够的空间

  • 这三个都有带n的版本strncmp,strncpy,strncat，安全版本

  • strchr从左边找，在字符串里寻找一个(s,3)，返回NULL表示没有找到

  • strrchr从右边找

  • strstr判断s2是不是s的子串，如果是，则该函数返回 s字符串从 s2第一次出现的位置开始到 s结尾的字符串；否则，返回NULL。
