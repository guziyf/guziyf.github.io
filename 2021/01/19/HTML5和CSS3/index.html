HTML5和CSS3

发表于 2021-01-19 更新于 2021-01-21 分类于 weblearn

HTML5的新特性

新增的语义化标签

  • <header>头部标签 <nav>导航标签 <article>内容标签 <section>定义文档某个区域 <aside>侧边栏标签 <footer>尾部标签
    • 语义化标签主要针对搜索引擎
    • IE9中，需要把这些元素转换成块级元素
    • 移动端更喜欢使用这些标签

新增的多媒体标签

  • 视频<video>
    • 支持mp4、ogg、Web
      • type="video/ogg"
    • 自动播放功能谷歌禁用autoplay
      • 解决方法：加静音播放muted="muted"
    • 展示播放控件controls
    • 设置宽高width height
    • 循环播放loop
    • 预加载preload:auto|none
      • 如果有autoplay，就忽视该属性
    • 加载等待画面图片poster:lmgurl
  • 音频<audio>
    • 有autoplay、controls、loop
    • 谷歌浏览器把音频和视频自动播放禁止了
      • 可以通过js解决

新增的input类型

  • type="email|url|data|time|month|week|number|tel|search|color"
    • 重点记住：number、tel、search

新增的表单属性

  • 必填required
  • 提示文字placeholder
  • 自动聚焦属性，就是文本输入符直接移动到指定表单autofocus
  • 当用户在字段开始键入时，浏览器基于之前键入过的值，应该显示出在字段中填写的选项，默认已经打开autocomplete="on"|"off"
  • 多选文件提交multiple

CSS3的新特性

  • ie9+才支持

CSS3新增选择器

  • 属性选择器（权重是10）

    • 不借助类或id<input type="text" value="请输入用户名">

      • 把value中的默认文字改成粉色：input[value] {color:pink}

    • 用=选择属性的值<input type="text">

      • 使输入的字体变粉色nput[type=text]{color:pink}

    • 可以选择属性值开头的某些元素

      • 用div[class^=icon] {color: red;}来修改

    • 以结尾写就用div[class$=data]

    • 只要有某个值就选择出来用div[class*=da]

  • 结构伪类选择器

    • ul li:first-child匹配父元素中的第一个子元素
    • last-child最后一个
    • nth-child(n)第n个
      • ul li:nth-child(2)选出来第2个
      • n可以是数字，关键字和公式，数字从1开始
      • 关键字可以是：even偶数，odd奇数
      • 只用n就是选择了所有孩子
      • 2n相当于偶数，2n+1相当于奇数
      • n+3，从第3个开始到最后
      • -n+3，从第3个到开始，包括第3个
    • section div:nth-of-type(1) {background-color: blue}当section中有p、div这些混用时，要选择特定元素类型，因为nth-child会给所有盒子都排序
      • div:first-of-type指定类型的第一个
      • div:last-of-type指定类型的最后一个

  • 伪元素选择器（权重1）

    • 创建的是行内元素

    <div>是</div>

    • div::before{content: '我'}打印在屏幕上就是“我是”

      • div::after{content: '小猪佩奇'}打印在“是”后面

    • 鼠标移动上去之后的遮罩，a:hover::before{}

    • 伪元素消除浮动

      • 额外标签法（空标签必须是块级）
      • 伪元素法
      • 双伪元素

盒子模型

  • box-sizing : content-box 默认就是这个
  • box-sizing : border-box最终盒子大小为width大小，padding和border就不会撑大盒子了（前提是padding和border不会超过width宽度）

其他特性

  • 图片变模糊
    • 滤镜filter，将模糊或颜色偏移等图形效果应用于元素
    • filter: blur(5px); blur模糊处理 数值越大越模糊
  • 计算盒子宽度width: calc函数
    • calc函数
    • width: calc(100% - 80px);
      • 里面的100%是相对于父盒子的宽度，-80px就是从右往左减

过渡

  • 从一个状态渐渐过渡到另一个状态
  • 经常和hover一起搭配
  • transition：要过渡的属性 花费时间(必须写，比如0.5s) 运动曲线 何时开始
  • 如果要多个属性变换，属性值用all就行，如transition: all .5s;

2D转换

  • transform:translate(x, y);

    • x是水平，y是垂直
    • 只动x轴transfrom : translateX(n)
    • 只动y轴transfrom : translateY(n)
    • 不会影响其他盒子的位置
      • 比如用在鼠标悬停时，盒子y轴像素位移
    • 使用%，就是用盒子自身对应的长度
    • 让盒子在父盒子中间水平居中

  • 旋转rotate

    • transform:rotate(度数)
    • 45deg顺时针旋转45度
    • 做三角箭头，正方形的两个边，加个旋转
    • 设置旋转中心点
      • transform-origin: x y;
      • 可以跟方位名词 left bottom

  • 缩放scale(x,y)

    • transform: scale(2,2)
      • 宽和高变成原来的两倍
      • 如果同时修改宽高，可以写成transform: scale(2);
    • 不会影响其他的盒子，而且可以设置中心点
      • 也可以设置中心点transform-origin: x y;
    • 鼠标经过放大效果

  • 同时使用多个转换时，顺序会影响转换的效果，先旋转会改变坐标轴位置

    • 当同时有位移和其他属性的时候，要将位移放到最前

动画

  • animation相比于过渡，动画可以实现更多变化，可以自己动
  • 用keyframes定义动画（类似定义选择器）
  • 实现盒子从左到右再从右到左的过程
  • 动画序列

  • 简写

    • animation: 动画名称 持续时间 运动曲线 何时开始 播放次数 倒放 动画起始或结束状态
    • 鼠标经过停止要单独拿出来写

  • 速度曲线的细节

    • linear匀速
    • ease默认
    • ease-in动画以低速开始
    • ease-out动画以低速结束
    • ease-in-out动画以低速开始和结束
    • steps()指定了时间函数中的间隔数量（步长）
      • 一帧一帧得走

3D转换

  • z轴，往外面时正值，往里面是负值
    • transform:translate3d(x,y,z)
  • 透视perspective
    • 写在被观察元素的父盒子上面
  • rotate3d
    • transform:rotate3d(x,y,z,deg)
  • 旋转方向
    • 左手拇指指向轴的正方向

3D呈现

  • transform-style:flat子元素不开启3d空间，默认

    • preserve-3d开启3d空间
    • 代码写给父级，但影响的是子盒子

  • 旋转木马
