12.移动web开发流式布局

发表于 2021-01-20 分类于 weblearn

移动端基础

  • 浏览器现状
    • 都是根据Webkit修改过来的内核
    • 兼容移动端主流浏览器，处理Webkit内核浏览器即可
  • 移动端调试方法
    • Chrome DevTools模拟手机调试
    • 本地web服务器，手机和服务器同一个局域网内通过手机访问服务器
    • 使用外网服务器，直接IP或域名访问

视口

  • 布局视口layout viewport

  • 视觉视口ideal viewport

    • 用户可以看见的网页区域
    • 写meta视口标签同志浏览器操作

  • meta视口标签

    • <meta name="viewport" content="width=device-width, user-scalable=no,initial-scale=1.0,maximum-scale=1.0,minimum-scale=1.0">

      | 属性 | 解释说明 |
      | ——————- | ———————————————— |
      | width | 宽度设置，可以设置成device-width |
      | initial-scale | 初始缩放比，大于0的数字 |
      | maximum-scale | 最大缩放比，大于0的数字 |
      | minimum-scale | 最小缩放比，大于0的数字 |
      | user-scalable | 用户是否可以缩放,yes或no（1或0） |

    • 标准的viewport设置

      • 视口宽度和设备保持一致
      • 视口的默认缩放比例1.0
      • 不允许用户自行缩放
      • 最大允许的缩放比例1.0
      • 最小允许的缩放比例1.0

二倍图

  • 像素比有变化
  • 准备的图片比实际需要的大小大2倍
  • 背景缩放background-size:500px 200px
    • 只写一个参数，会等比例缩放
    • 用百分比是相对于父盒子
    • cover图片覆盖盒子，图片宽高比例不变，可能有部分背景图片超出
    • contain等比例扩大至最多，在宽和高都不超出父盒子的前提下铺满
