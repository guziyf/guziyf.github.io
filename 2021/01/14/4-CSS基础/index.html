4.CSS基础

发表于 2021-01-14 更新于 2021-01-18 分类于 weblearn

1.引入方式

  • 行内样式、内部样式

    • 行内样式通过HTML的style属性引入
    • 局限性：只对当前的元素生效
    • 行内的优先级更高
    • 内部样式
  • 外部样式
    • 新建css文件
    • head中用link标签引入
      • rel告诉引入的是什么，href告诉什么事路径
    • 实现了内容和表现的分离，提高了代码的可重用性和可维护性
  • 内部样式和外部样式优先级一样，谁写在后面谁生效

2.import和link的区别

  • 加载顺序不同，@import 先加载HTML文件，再加载css文件，link是同时加载HTML和css文件
  • 加载内容不同，@import只能引入css文件，link还可以引入其他内容
    • <link rel = “icon” href = “图片路径” >
  • 兼容性不同，@import有兼容性（IE5以上支持），link没有兼容性
  • @import会增加页面的http请求
  • js操作DOM时，只能操作link引入的css

3.基础选择器

  • 全局选择器、通用选择器

  • 优先级：行内样式>ID选择器>类选择器>元素选择器>全局选择器

    • 权重，按顺序来1000、100、10、1

  • ID选择器

  • HTML的四大通用属性
    • title 鼠标悬停给予提示
    • style 引入css的行内样式
    • class 类选择器
    • id ID选择器

4.字体属性

  • 字体颜色
    • color
  • 字体大小
    • font-size:16px;
    • 默认字体大小16px
    • 支持最小12px
  • 字体是否加粗
    • font-weight:;
    • normal 默认值，不加粗
    • bold 加粗
    • 100-900 400=normal 700=bold
  • 字体倾斜
    • font-style:normal|italic;
  • 字体
    • font-family:字体1,字体2,…;
    • 当字体包含空格时，必须加引号，比如”Microsoft YaHei”

5.文本属性

  • 水平对齐方式
    • text-align:left|right|center;
  • 文本装饰
    • text-decoration:underline|none;下划线

  • 英文字母大小写转换

    • text-transform:;
    • capitalize 每个单词首字母大写
    • uppercase 全部大写
    • lowercase 全部小写

  • 首行缩进

    • text-indent:;
    • 单位：px像素 绝对单位
    • em 相对单位 1em=16px
  • 行间距
    • line-height
    • 行间距包含了上间距+文本高度+下艰巨，多出来的像素上下平分

5.背景属性

  • background-color

    • transparent（透明的）

  • background-image:url(“”);

  • background-repeat

    • 背景图片是否平铺
      • repeat默认值，平铺
      • no-repeat不平铺
      • repeat-x水平方向平铺
      • repeat-y垂直

  • background-size

    • 100% 100%
    • 1000px 500px
    • cover背景图片可能显示不完全
    • contain背景图片显示完全

  • background-position

    • 图片定位，默认在左上角
    • 取值：px % left right center top bottom
    • 如果只写了x轴，第二个值默认center

  • background-attachment

    • 背景图片固定住，不随滚动移动
    • scroll默认值|fixed固定

  • 简写

    • background:red;
    • background:url(“”) no-repeat;
    • 当简写属性和单个属性同时存在，简写属性要写在单个属性的上面
    • 一般约定的顺序是：背景颜色、背景图片地址、背景平铺、背景图像滚动、背景图片位置

5.列表属性

  • list-style-type: none取消列表前的点
  • list-style-image: url(“images/favicon.ico”);将点换成icon
  • 设置项目符号的位置list-style-position: outside|inside;
  • 简写list-style: none;

6.表格属性

  • 表格边框
  • 元素内容水平对齐方式
    • text-align:left|center|right
  • 表格被绒的垂直对齐方式
    • vertical-align:middle|top|bottom
  • 边框折叠
    • 相当于cellspacing=”0”
    • border-collapse:collapse;

7.其他属性

  • 内容溢出

    • overflow: hidden;溢出部分隐藏掉
    • auto自动生成滚动条
    • scroll不管内容是否溢出，都显示滚动条

8.复合选择器

8.1 关系选择器

  • 后代选择器

    • 用空格，选择所有后代，包括儿子

  • 子代选择器

    • 用>，只有儿子

    • 能用子代先用子代

  • 链接伪类选择器

    • 注意：必须按顺序来，比如把hover放link前面就失去效果了

  • focus伪类选择器

    • 获得焦点就是光标

  • 相邻兄弟选择器

    • +仅下一个

  • 通用兄弟选择器

    • ~后面所有

9.CSS的三大特性

层叠性

  • 样式冲突，就近原则

继承性

  • 只标签会继承父标签中的某些样式

  • text-，font-，line-，color这些元素开头可以继承

  • 行高的继承

优先级

选择器 选择器权重
继承 或者 * 0,0,0,0
元素选择器 0,0,0,1
类选择器，伪类选择器 0,0,1,0
ID选择器 0,1,0,0
行内样式 style=”” 1,0,0,0
!important 重要性 ∞ 无穷大
  • 如果是复合选择器，则会有权重叠加，需要计算权重
    • div ul li ———-> 0,0,0,3 三个元素选择器
    • .nav ul li ——-> 0,0,1,2 一个类加两个元素
    • a:hover ———> 0,0,1,1 元素加伪类
    • .nav a ————> 0,0,1,1 类加元素
  • 先看有没有标签单独选出来修改样式，如果选出来了，优先执行选出来的样式

10.盒模型

10.1 概念

  • 包括外边距margin、边框border、内边距padding、内容content
  • 边框可以分开写

10.2 表格的细线边框(合并边框)

  • border-collapse

10.3 边框会影响盒子实际大小

  • 测量盒子时不要边框

10.4 内边距(padding)

  • 简写法
值的个数
padding:5px 上下左右5
5px 10px 上下5，左右10
5px 10px 20px 上5，左右10，下20
5px 10px 20px 30px 上5，右10，下20，左30
  • 内容和边框有了距离，添加了内边距
  • 影响盒子实际大小，如果盒子有宽度和高度，再指定内边框，会撑大盒子
  • 如果不指定宽度，没必要写100%

10.5 外边距(margin)

  • 基本写法和padding一致

  • 外边距可以让块级盒子水平居中，但必须满足两个条件

    • 盒子必须指定了宽度
    • 盒子的左右外边距设置为auto

  • 行内元素给他们的父元素添加text-align:center 即可

  • 盒子内的上边距用margin来做

  • 父子盒子嵌套用padding好，相邻用margin好

10.6 外边距合并

  • 对于两个嵌套关系的块元素，两者同时有上边距时，父元素会塌陷较大的外边距值(父与子谁大按照谁的来)
    • 为父元素定义上边框
    • 为父元素定义上内边距
    • 为父元素添加overflow:hidden

10.7 清除内外边距

  • 很多元素有默认的内外边距，布局前，先清除
  • 行内元素为了照顾兼容性，尽量只设置左右内外边距，不要设置上下内外边距。但是转换为块级和行内块元素就可以了

11.圆角边框

  • border-radius: 20px;
    • 画一个半径为20px的圆，和矩形相切截
  • border-radius: 50%;
    • 高度的一半
  • 跟四个值，分别代表左上角、右上角、右下角、左下角
  • 也可以border-top-left-radius:;
    • 注意要先写top或bottom

12.盒子阴影

属性
h-shadow 必需，水平位置，正负
v-shadow 必需，垂直位置，正负
blur 模糊距离，糊的程度
spread 阴影尺寸
color 阴影颜色
inset 改为内阴影

13.文字阴影

  • text-shadow
    • 和盒子阴影前4个一样

14.CSS属性书写顺序（重点）

1 布局定位属性

display/position/float/clear/visibility/overflow

2 自身属性

width/height/margin/padding/border/background

3 文本属性

color/font/text-decoration/text-align/vertical-align/white-space/break-word

4 其他属性(CSS3)

content/cursor/rgba/border-radius/box-shadow/text-shadow/background:linear-gradient

15.元素的显示和隐藏

  • display显示隐藏
    • display: none
    • 盒子消失，不再占有原来的位置
  • visibility显示隐藏
    • inherit|visible|collapse|hidden
    • hidden隐藏后位置保留
  • overflow溢出显示隐藏
    • hidden在盒子内隐藏
    • scroll滚动条，不管超出没超出都显示滚动条
    • auto在需要的时候添加滚动条
