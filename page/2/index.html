html基础

发表于 2021-01-13 更新于 2021-01-21 分类于 weblearn

各种属性查阅文档

标签名 定义 说明
html HTML标签 最大的标签，可称根标签
head 文档头部 head中必须要设置titile
titile 文档标题 网页的标题
body 文档主体 元素包含文档所有内容，页面内容基本都在body
  • 标题标签
  • 段落标签
    • 分段就必须用段落标签
  • 换行标签
    • 单标签
  • 文本格式化标签
语义 标签 说明
加粗 strong或者b 更推荐使用strong
倾斜 em或者i 更推荐em
删除线 del或者s 更推荐del
下划线 ins或者u 更推荐ins
  • div和span标签
    • 没有语义
    • 就是一个盒子，用来装内容
  • 图像标签
属性 属性值 说明
src 图片路径 必须属性
alt 文本 替换文本，图像不能显示的文字
title 文本 提示文本，鼠标放在图像上，显示文字
width 像素 设置图像宽度
height 像素 高度
border 像素 图像边框粗细
  • 超链接标签
属性 作用
href url地址
target 默认为 _self ，新窗口打开是 _blank
空链接 a href = “#”
锚点链接 href = “#名字”，对应位置加id = “名字”

表格标签

属性名 属性值 描述
align left、center、right 规定表格相对周围元素的对齐方式
border 1或”” 规定表格单元是否拥有边框，默认为””，表示没有边框
cellpadding 像素值 规定单元边沿与其内容之间的空白，默认1像素
cellspacing 像素值 规定单元格之间的空白,默认2像素
width 像素值或百分比 规定表格的宽度

  • 表格结构标签

    • thead定义表格头部
    • tbody定义表格主体，主要用于放数据本体

  • 合并单元格

    • 跨行合并：rowspan=””
    • 跨列合并：colspan=””

列表标签

  • 无序列表
  • 去掉小圆点：list-style:none
  • 有序列表
  • 自定义列表

元素显示模式

  • 块级元素
    • div独占一行
    • 是一个容器及盒子
    • 文字类的元素内不能放块，比如p、h1

  • 行内元素

    • span、strong、ins等等
    • 宽高直接设置是没有效果的
    • 默认宽度就是它本身内容的宽度
    • 只能容纳文本和其他行内元素

  • 行内块元素

    • img、input、td
    • 和相邻行内元素在一行时，之间会有空隙
  • 转换模式
    • block|inline|inline-block

form

  • form之间不能相互嵌套

用户名和密码

单选和多选

  • label提供点击文字选择功能

  • input type=”file”默认的表单编码格式不支持上传文件，但表单元素有文件时，需要把表单的编码格式改为二进制enctype

表单元素的属性

元素
type 设置输入框
name 表单元素名称
value 当前值
*checked 默认被选中，一半和按钮一起用
selected 默认显示，配合option
disable 禁用，属性等于属性值
readonly 只读,可以点击但不能编辑，属性等于属性值
